# 🎬 AI Movie Recommendation System

An interactive web-based Movie Recommendation System that suggests movies based on user-selected genres and allows direct movie search with detailed information fetched from the OMDb API.

## 📌 Project Overview

This project helps users discover movies according to their interests by selecting genres such as Action, Drama, Comedy, Thriller, Adventure, and more.

The application ranks movies based on genre matching and displays:

- Movie posters
- IMDb ratings
- Release year
- Plot summary
- Match score
- Trailer search link

Users can also search for any movie directly and view its details instantly.

## ⚙️ How It Works

### Genre Recommendation Algorithm

1. User selects one or more genres.
2. The system compares selected genres with each movie's genre list.
3. A match score is calculated.
4. Movies are sorted based on their score.
5. The highest-ranked movie is displayed as the Top Recommendation.

### Movie Search

1. User enters a movie title.
2. The application sends a request to the OMDb API.
3. Movie information is fetched and displayed.
