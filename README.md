# DecodeLabs-Intership-projects

This repository contains the projects completed during the DecodeLabs Internship Program.

## Projects Included

### 1. Rule-Based AI Chatbot

A chatbot developed using predefined rules and responses using HTML, CSS, and JavaScript.

### 2. Data Classification Using AI

A machine learning project that classifies data into different categories based on input features.

### 3. AI Movie Recommendation System

A recommendation system that suggests movies based on user preferences and similarity analysis.

### 4. Image-to-Text Recognition

An OCR project that extracts text from images using OpenCV and Tesseract OCR.

## Technologies Used

* Python
* Machine Learning
* OpenCV
* Tesseract OCR
* HTML
* CSS
* JavaScript
* Google Colab
