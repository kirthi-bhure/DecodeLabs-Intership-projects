# 🤖 NEXUS-9 Rule-Based AI Chatbot

A futuristic Rule-Based AI Chatbot built using HTML, CSS, and JavaScript. The project features a modern cyberpunk-inspired interface, glassmorphism effects, responsive design, and predefined chatbot responses based on user input.

---

## 📌 Project Overview

NEXUS-9 is a front-end chatbot application designed to demonstrate the fundamentals of rule-based conversational systems. Unlike AI-powered chatbots, it responds using predefined rules and command mappings. The project showcases JavaScript event handling, DOM manipulation, responsive web design, and user-friendly chatbot interactions.

## ⚙️ How It Works

1. User enters a message in the chat box.
2. JavaScript captures the input.
3. The chatbot compares the input against predefined rules.
4. If a matching command is found, the corresponding response is displayed.
5. If no match exists, a default response is shown.
6. Messages are displayed with timestamps for better interaction.

### Example Commands

| User Input | Bot Response |
|------------|-------------|
| hello | Greetings, User! |
| hi | Hello! How can I assist you today? |
| help | Displays available commands |
| how are you | My systems are functioning optimally. |
| bye | Goodbye! Have a great day. |
