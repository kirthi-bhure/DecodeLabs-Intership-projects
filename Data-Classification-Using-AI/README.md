# Iris Classification Project

## Project Title
Data Classification Using AI using Iris Dataset

## Objective
The objective of this project is to build a machine learning classification model that predicts flower species using the Iris dataset.

## Tools and Libraries Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib

## Dataset Information
The Iris dataset contains:
150 samples
4 features:
Sepal Length
Sepal Width
Petal Length
Petal Width
3 classes:
Setosa
Versicolor
Virginica

## Steps Performed
1. Loaded the Iris dataset
2. Converted data into a Pandas DataFrame
3. Split features and target values
4. Divided data into training and testing sets
5. Applied StandardScaler for feature scaling
6. Trained the KNN classification model
7. Made predictions on test data
8. Evaluated model using:
   Accuracy Score
   Confusion Matrix
   F1 Score

## Model Used
K-Nearest Neighbors (KNN)

## Why StandardScaler?
StandardScaler is used to normalize feature values so that KNN distance calculations work correctly and no feature dominates due to larger values.

## Results
High Accuracy achieved
Confusion Matrix generated
F1 Score calculated

## Conclusion
The Iris flower classification model was successfully built using supervised learning. The KNN algorithm performed well in classifying flower species accurately.
